# pw-16-damiano-francesco-web-app

### Come avviare l'applicazione

0. Inizializzare l'ambiente rinominando il file .env.example in .env
1. Avere docker installato e a portata di mano
2. Inserire il seguente comando nel terminale:
`docker compose up --build`

3. Dopo aver eseguito questo comando, docker provvederà ad installare tutti i servizi necessari per poter avviare l'applicazione.
4. Saranno inoltre eseguiti script .sql, presenti nella cartella /Backend/Database, che inseriscono dati di test nel database e creano schemi, relazioni, vincoli, indici.

5. Et voilà, Dottori & dolori è servita. (Niente di più semplice)
